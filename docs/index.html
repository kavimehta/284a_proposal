Title

Filling Holes in Point Clouds with GANs

Team

Kavi Mehta, 25040386

Jack Sullivan, 25195800

David Tseng, 25291067

Summary

We are trying to solve the problem of creating complete meshes when there are holes in the point cloud data. There are existing projects that accomplish this using heuristic methods, as well as methods that reconstruct incomplete 2D images using GANs. We propose a combination of these ideas: using GANs to fill holes in point clouds in order to reconstruct a complete mesh.
